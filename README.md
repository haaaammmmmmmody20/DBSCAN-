# DBSCAN Algorithm

This project implements the DBSCAN (Density-Based Spatial Clustering of Applications with Noise) algorithm.

## About
DBSCAN is a clustering algorithm used in data analysis and machine learning.  
It groups points that are close to each other and marks points that are far away as noise.

## Features
- Detects clusters based on density
- Can find arbitrary shaped clusters
- Handles noise (outliers)

## How it works
The algorithm uses two main parameters:
- eps: the maximum distance between two points
- minPts: the minimum number of points to form a cluster
